WebMonX 0.3.3
http://debfx.fobos.de/webmonx/
-------------------------------

WebMonX monitors websites for updates and changes - it saves you time and
always keeps you up-to-date by periodically checking these pages at set intervals.
WebMonX is free/open-source software (licensed under GPLv2/GPLv3) and is a
cross-platform (works on *nix and Windows) port of WebMon.
